## Documentation de Elixir BOT

###Liste des commandes

Prefix: **pdp.**

- **random** Permet d'envoyer une pdp au hasard
- **normal** Permet d'envoyer une pdp (png, jpg)
- **gif** Permet d'envoyer une pdp (gif)
- **upload** Permet d'upload des images et gif dans notre database
- **info** Permet de voir les infos du bot
- **ping** Permet de voir le ping du bot
- **reportbug** Permet de report des bugs
